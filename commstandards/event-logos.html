Logo Guidelines

The following guidelines will help you select the most appropriate logo format for your project.

How To Select a Logo Format

For most printed material, use high-resolution EPS files. For some publications, low-resolution JPG files may be acceptable. For Web graphics or photos up to 200 pixels, use GIFs. For photos more than 200 pixels, use JPGs.

How To Download Logos

To download an EPS file, click on the appropriate link below and save the file to your computer. To decompress and use EPS files, they must be opened in an application such as Photoshop. To download a GIF or JPG, click on the appropriate link below to open the file in a new window. Select the image and use the "Save Picture As" function to save the image to your computer. Make sure you save the file in the correct file format (e.g., .eps, .jpg, or .gif).
